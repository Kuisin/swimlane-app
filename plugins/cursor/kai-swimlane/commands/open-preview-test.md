# Open preview test

If the Kai Swimlane VS Code extension is installed:

1. Open `extensions/vscode-kai-swimlane/samples/preview-test.md` from the kai-swimlane repository.
2. Run **Markdown: Open Preview** (`Cmd+Shift+V` / `Ctrl+Shift+V`).
3. Confirm `kai-swimlane` and `kai-swimlane-parts` fences render as diagrams.

If preview shows plain code blocks, run the install script from the Cursor plugin README or install the `.vsix` from GitHub Releases.
